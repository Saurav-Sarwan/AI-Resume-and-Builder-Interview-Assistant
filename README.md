# AI Resume Builder & Interview Assistant

An AI-powered full-stack web application that analyzes your resume and a target job description to generate a personalized interview strategy вЂ” including technical and behavioral questions, skill gap analysis, a day-wise preparation plan, and a tailored resume PDF.

---

## Features

- **AI Interview Report** вЂ” Upload your resume and paste a job description to get a full interview strategy generated by Google Gemini AI
- **Match Score** вЂ” See how well your profile matches the job (0вЂ“100%)
- **Technical Questions** вЂ” 7 role-specific questions with interviewer intent and suggested answers
- **Behavioral Questions** вЂ” 3 STAR-method questions with guidance on how to answer
- **Skill Gap Analysis** вЂ” Identifies missing skills with severity ratings (low / medium / high)
- **Day-wise Prep Plan** вЂ” A structured daily preparation schedule tailored to the role
- **AI Resume PDF** вЂ” Generate a job-tailored, ATS-friendly resume as a downloadable PDF
- **Authentication** вЂ” JWT-based auth with cookie storage and token blacklisting on logout
- **Report History** вЂ” View all previously generated interview reports

## API Reference

### Auth Routes вЂ” `/api/auth`

| Method | Endpoint | Access | Description |
|---|---|---|---|
| POST | `/register` | Public | Register a new user |
| POST | `/login` | Public | Login and receive JWT cookie |
| GET | `/logout` | Public | Clear cookie and blacklist token |
| GET | `/get-me` | Private | Get current logged-in user |

### Interview Routes вЂ” `/api/interview`

| Method | Endpoint | Access | Description |
|---|---|---|---|
| POST | `/` | Private | Generate interview report (multipart/form-data) |
| GET | `/` | Private | Get all reports for logged-in user |
| GET | `/report/:interviewId` | Private | Get a specific report by ID |
| GET | `/resume/pdf/:interviewReportId` | Private | Download tailored resume as PDF |

## Getting Started

### Prerequisites

- Node.js v18+
- MongoDB instance (local or Atlas)
- Google Gemini API key
- Chromium/Chrome installed (for Puppeteer PDF generation)

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-resume-builder.git
cd ai-resume-builder
```

### 2. Backend setup

```bash
cd Backend
npm install
```

Create a `.env` file in the `Backend/` directory:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
GOOGLE_GENAI_API_KEY=your_google_gemini_api_key
PUPPETEER_EXECUTABLE_PATH=/path/to/chrome  # optional, leave blank to use default
```

Start the backend server:

```bash
node server.js
```

The server runs on `http://localhost:3000`.

### 3. Frontend setup

```bash
cd Frontend
npm install
npm run dev
```

The frontend runs on `http://localhost:5173`.

## Usage

1. **Register** a new account or **Login** with existing credentials
2. On the home page, paste the **job description** of the role you're targeting
3. Either **upload your resume** (PDF) or write a **quick self-description**
4. Click **Generate My Interview Strategy** and wait ~30 seconds for the AI to process
5. Review your personalized report вЂ” questions, skill gaps, match score, and prep plan
6. Download a **tailored resume PDF** directly from the report page
7. Access all your past reports from the home page dashboard
